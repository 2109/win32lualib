Detailed Description

Generic header to be included when handling errors.

When the library is compiled with OOLUA_USE_EXCEPTIONS == 1 it will include the oolua_exception.h header and provide dummy implemenations for OOLUA::get_last_error and OOLUA::reset_error_value. When compiled with OOLUA_STORE_LAST_ERROR == 1 it provides implements for OOLUA::get_last_error and OOLUA::reset_error_value.
