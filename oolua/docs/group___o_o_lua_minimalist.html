Detailed Description

Generates code using the minimal information.

Generates a proxy function using the only the minimal of information which is generally the name of the thing being proxied and possibly a new name for the proxy. As with taking the address of any C++ function, if there is any ambiguity it will fail to compile, in which case a user should help the compiler by specifying more information using the matching, yet longer named Expressive DSL entry.
The longer DSL name requires more information.

Note
No Traits can be expressed with this DSL group.
